# Student-Performance-Analysis
📊 Student Performance Analytics Dashboard (Power BI)
📌 Project Overview

This project presents a data analytics dashboard built using Microsoft Power BI to analyze student academic performance and behavioral factors.

The dataset was enhanced by creating multiple calculated columns and derived performance indicators to generate deeper analytical insights beyond the raw data.

🛠 Tools & Technologies

Microsoft Power BI

DAX (Calculated Columns & Measures)

Data Cleaning & Transformation

Excel (for dataset export)

GitHub

📂 Dataset Description

The dataset contains student-related attributes such as:

Demographic information

Study habits

Social and behavioral factors

Academic performance indicators

All columns after “romantic” were custom-calculated to enhance analytical depth. These derived fields include performance classifications, aggregated scores, and performance-based metrics.

🔎 Data Processing & Transformations

The following steps were performed:

Cleaned and structured raw student data

Created calculated columns using DAX

Derived new performance indicators

Categorized students based on academic metrics

Built summary measures for dashboard KPIs

📊 Dashboard Features

Student performance distribution

Comparison across study habits and lifestyle factors

Performance categorization

Derived performance scoring metrics

Interactive filtering and slicers

📈 Key Insights

Identified patterns between study behavior and academic results

Analyzed impact of lifestyle and social factors on performance

Segmented students based on calculated performance indicators

Highlighted trends across multiple variables
